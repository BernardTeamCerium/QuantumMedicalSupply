# Accreditation / Trust Badges

The homepage shows an "Accredited & trusted" row of logo tiles. The files here
are **placeholders** so nothing false is published.

## Replace with your real seals
Drop your actual accreditation logos in this folder using the same filenames
(PNG with a transparent or white background works best, roughly 300×140):

- `accreditation-1.png`
- `accreditation-2.png`
- `accreditation-3.png`
- `accreditation-4.png`

Common examples for DME suppliers: **ACHC** (or other accreditation body),
**Medicare-enrolled supplier**, **BBB**, **HIPAA compliance**. Only display
seals you are actually authorized to use.

Have fewer or more than four? Tell me and I'll adjust the row, alt text, and
spacing to match.
